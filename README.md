## 主要方法（main method）
* 聚类分析（Cluster analysis）
* RFM模型（RMF Model）
* * RFMCL
```
R:上次购票距今的时间
F:飞行频次
M:飞行公里数
C:综合的折扣率
L:入会的时间
```
